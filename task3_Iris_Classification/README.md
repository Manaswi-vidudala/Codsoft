The third project is Iris Flower Classification.
The objective of this project is to classify iris flowers into three species: Setosa, Versicolor, and Virginica.
The dataset contains four main features: sepal length, sepal width, petal length, and petal width.
I used Logistic Regression to build the classification model.
First, I loaded the dataset using Pandas, then separated the input features and target variable.
After that, I split the data into training and testing sets and trained the model.
Finally, I evaluated the model using accuracy score.
The model achieved 100% accuracy, meaning it correctly classified all the test samples.
