The fifth project is Credit Card Fraud Detection.
The objective of this project is to identify fraudulent credit card transactions.
The dataset contains transaction details such as time, amount, and several anonymized features.
One challenge in this dataset is that it is highly imbalanced, meaning fraudulent transactions are very rare compared to normal transactions.
I performed preprocessing and feature scaling, then split the dataset into training and testing sets.
I trained classification models such as Logistic Regression and Random Forest.
The model was evaluated using metrics such as precision, recall, and F1-score, which are important for fraud detection problems.
