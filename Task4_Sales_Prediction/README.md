The fourth project is Sales Prediction using Machine Learning.
The objective of this project is to predict product sales based on advertising budgets.
The dataset includes advertising spending on TV, radio, and newspaper platforms.
I used a Linear Regression model because sales is a continuous numerical value.
After loading the dataset, I analyzed the relationship between advertising expenditure and sales.
Then I trained the model using training data and evaluated it using the R-squared score.
The results showed a strong relationship between advertising spending and product sales.
