The first project is Titanic Survival Prediction.
The goal is to predict whether a passenger survived the Titanic disaster.
The dataset includes features such as passenger class, gender, age, fare, and embarkation port.
I performed data preprocessing by handling missing values and converting categorical variables into numerical values.
Then I split the dataset into training and testing sets.
I trained a Logistic Regression model and also experimented with a Random Forest model to improve performance.
The model achieved around 80 percent accuracy in predicting passenger survival
