The second project is Movie Rating Prediction.
The goal is to predict the rating of a movie based on its characteristics such as genre, director, and actors.
The dataset contained categorical variables, so I converted them into numerical values using encoding techniques.
After preprocessing the data, I split it into training and testing sets.
I used a Random Forest Regression model to predict movie ratings.
The model was able to estimate movie ratings with reasonable accuracy and helped identify the factors that influence movie ratings
